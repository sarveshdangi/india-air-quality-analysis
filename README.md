# 🌍 India Air Quality Analysis & Prediction

## 📌 Overview
This project analyzes air quality data across India and predicts pollution levels using Machine Learning.

## 🔍 Features
- Data Cleaning
- Data Visualization
- Pollution Analysis
- Linear Regression Model

## 📊 Key Insights
- PM2.5 & PM10 contribute most pollution
- NCR region is highly polluted
- Strong correlation between pollution variables

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 🚀 How to Run
pip install -r requirements.txt  
python src/project.py

## 📁 Project Structure
data/ → dataset  
src/ → code  
outputs/ → graphs
